## <a name="conventions"></a><span data-ttu-id="f6cef-329">Соглашения</span><span class="sxs-lookup"><span data-stu-id="f6cef-329">Conventions</span></span>

<span data-ttu-id="f6cef-330">Объем кода, нужно было написать в платформе Entity Framework иметь возможность создать для вас всей базы данных сводится к минимуму из-за *соглашения*, или предположения, сделанные в Entity Framework.</span><span class="sxs-lookup"><span data-stu-id="f6cef-330">The amount of code you had to write in order for Entity Framework to be able to create a complete database for you is minimal because of *conventions*, or assumptions that Entity Framework makes.</span></span> <span data-ttu-id="f6cef-331">Некоторые из них уже было отмечено, или был использован без вашего сведений о них:</span><span class="sxs-lookup"><span data-stu-id="f6cef-331">Some of them have already been noted or were used without your being aware of them:</span></span>
